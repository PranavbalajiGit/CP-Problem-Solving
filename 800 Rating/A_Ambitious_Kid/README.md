# 1866A -- Ambitious Kid

## 1. Problem Statement

You are given an array (A) of (N) integers.\
In one operation, you may increase or decrease any element by 1.\
Your goal is to make the **product of all elements equal to zero**.\
Find the **minimum number of operations** required.

------------------------------------------------------------------------

## 2. Input / Output Format

### Input

-   First line: integer (N) --- number of elements\
-   Second line: (N) space-separated integers (A_1, A_2, `\dots`{=tex},
    A_N)

### Output

-   A single integer: the **minimum operations** needed to make the
    product zero

------------------------------------------------------------------------

## 3. Constraints

-   (1 `\le `{=tex}N `\le 10`{=tex}\^5)\
-   (-10\^5 `\le `{=tex}A_i `\le 10`{=tex}\^5)

------------------------------------------------------------------------

## 4. Examples

### Example 1

    Input:
    3
    2 -6 5
    Output:
    2

### Example 2

    Input:
    1
    -3
    Output:
    3

### Example 3

    Input:
    5
    0 -1 0 1 0
    Output:
    0
